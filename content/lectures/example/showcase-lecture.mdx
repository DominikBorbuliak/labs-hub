## Blockquotes

> A blockquote is useful for calling out quoted material or general emphasis. You can include **bold**, _italic_, and `code` inside blockquotes.

## Alert Boxes

Five alert variants are available using GitHub-style blockquote syntax.

> [!NOTE]
> Use notes to highlight information readers should be aware of, even when skimming.

> [!TIP]
> Tips provide optional advice that helps the reader be more successful.

> [!IMPORTANT]
> Important alerts convey crucial information necessary for the reader to succeed.

> [!WARNING]
> Warnings demand immediate attention due to potential risks or pitfalls.

> [!CAUTION]
> Caution alerts describe negative consequences of a particular action.

## Code Blocks

Syntax-highlighted code blocks are supported for several languages.

```typescript
interface Lecture {
  title: string;
  order: number;
  tags: string[];
}

function summarize(lecture: Lecture): string {
  return `${lecture.title} (${lecture.tags.join(", ")})`;
}
```

```python
from dataclasses import dataclass

@dataclass
class Lecture:
    title: str
    order: int
    tags: list[str]

    def summarize(self) -> str:
        return f"{self.title} ({', '.join(self.tags)})"
```

```csharp
public record Lecture(string Title, int Order, string[] Tags)
{
    public string Summarize()
        => $"{Title} ({string.Join(", ", Tags)})";
}
```
