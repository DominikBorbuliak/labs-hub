# Activity 2: Building a Console Memory Game (Pexeso)

## Step 1: Create the Project

Create a new console application project in .NET using C#.

## Step 2: Read User Inputs

- Read two numbers from the user
- If their product is **even**, print `OK`

## Step 3: Create the Card Class

Create a `Card` class that represents a single pexeso card in the game.

## Step 4: Create the Board Class

Create a `Board` class that represents the game board:

- Contains a `Start()` method that launches the game
- At the beginning, read two numbers from the player
  - The **product of these numbers must be even** (determines board size)
- Display the board with cards face-down:

```plain
[*][*][*][*]
[*][*][*][*]
[*][*][*][*]
```

## Step 5: Initialize Cards

Fill the cards with letters from the alphabet based on the board size (each letter appears twice for matching pairs).

## Step 6: User Input and Card Flipping

- Get user input (which cards to flip)
- Flip the selected cards
- Update and display the board state

## Step 7: Complete the Game

Implement full memory game logic for a single player:

- Matching pairs
- Win condition

## Step 8: Randomization

Modify card generation to **randomize** card placement on the board.
