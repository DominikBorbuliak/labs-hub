# LINQ

LINQ (Language Integrated Query) provides a unified syntax for querying and transforming data. Common flavors:

- LINQ to Objects (`IEnumerable<T>`)
- LINQ to Entities (`IQueryable<T>`)
- PLINQ
- LINQ to JSON, XML, etc.

## Example — LINQ

```csharp
using System;
using System.Linq;
using System.Collections.Generic;

var query = new[] { 1, 2, 3, 4, 5, 6 }
    .Where(x =>
    {
        Console.WriteLine($"Where({x})");
        return x % 2 == 0;
    })
    .Skip(1)
    .Select(x =>
    {
        Console.WriteLine($"Select({x})");
        return x * 2;
    })
    .Take(2);

Console.WriteLine("Enumerating query...");

foreach (var v in query)
{
    Console.WriteLine($"Received: {v}");
}
```

## Example — yield-return

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

IEnumerable<int> DeferredGenerator()
{
    Console.WriteLine("DeferredGenerator() started");

    for (int i = 1; i <= 5; i++)
    {
        Console.WriteLine($"Yielding {i}");
        yield return i;
    }

    Console.WriteLine("DeferredGenerator() finished");
}

var deferred = DeferredGenerator()
    .Where(x => x % 2 == 1);

foreach (var v in deferred)
{
    Console.WriteLine($"Received {v}");
}
```

### Example: MyApp.Dal

<Spoiler label="Double click to reveal the example solution">

```csharp
// MyApp.Dal/Entities/Person.cs
namespace MyApp.Dal.Entities
{
    public class Person
    {
        public int Id { get; set; }
        public string Name { get; set; }
        public int Age { get; set; }
    }
}

// MyApp.Dal/AppDbContext.cs
using Microsoft.EntityFrameworkCore;
using MyApp.Dal.Entities;

namespace MyApp.Dal
{
    public class AppDbContext : DbContext
    {
        public DbSet<Person> People { get; set; }

        public AppDbContext(DbContextOptions<AppDbContext> options) : base(options)
        {}

        protected override void OnModelCreating(ModelBuilder modelBuilder)
        {
            modelBuilder.Entity<Person>().HasData(
                new Person
                {
                    Id = 1,
                    Name = "Alice",
                    Age = 30
                },
                new Person
                {
                    Id = 2,
                    Name = "Bob",
                    Age = 25
                },
                new Person
                {
                    Id = 3,
                    Name = "Carol",
                    Age = 28
                }
            );
        }
    }
}
```

</Spoiler>

### Example: MyApp.Console

<Spoiler label="Double click to reveal the example solution">

```csharp
// MyApp.Console/Program.cs
using System;
using System.Linq;
using Microsoft.EntityFrameworkCore;
using MyApp.Dal;
using MyApp.Dal.Entities;

var options = new DbContextOptionsBuilder<AppDbContext>()
    .UseSqlite("Data Source=lecture10.db")
    .Options;

using var db = new AppDbContext(options);

db.Database.EnsureDeleted();
db.Database.EnsureCreated();

var all = db.People.ToList();

Console.WriteLine("People in database:");

foreach (var p in all)
{
    Console.WriteLine($"{p.Id}: {p.Name} ({p.Age})");
}
```
